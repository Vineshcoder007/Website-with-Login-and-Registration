# Login & Registration Page
## Project Overview

This project is a Login and Registration Page developed as part of the Ayu-Heal collaborative project. 
Ayu-Heal was a mini project created through team collaboration, and this module was designed to handle user authentication.

## My Contribution

I developed the Login and Registration system for the Ayu-Heal project. This module allows users to:

- Register with their details

- Log in using valid credentials

- Access the system securely

## Purpose of This Module

The login and registration page was built to:

- Provide secure user authentication

- Manage user access

- Serve as the entry point for the Ayu-Heal mini project

## Features

- User Registration Form

- User Login Form

- Input Validation

- Basic Authentication Handling
